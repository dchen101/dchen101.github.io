React Launchpad

Link: https://powerful-temple-9048.herokuapp.com/

Repo: https://github.com/dchen101/React-Launchpad-deploy

Details:

The project was built with React and Ruby on Railsails. React was used to render each individual tile with its own sound, and property changes. The Rails framework was used to manage the asset pipeline. The idea was to create a virtual launchpad web app for music enthusiasts who want to experience mixing sounds without the cost of purchasing a launchpad. Features to be added in the future will be hte bility for users to add their own music files to the asset pipeline
